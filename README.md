# Blog UI Project

## Overview

This is a simple front-end blog interface built with **HTML, CSS, and JavaScript**. The project focuses on page layout, responsive design, basic interactivity, and clean presentation of blog-style content.

The current version is a static/front-end project. Backend features such as authentication, database storage, comments, and full CRUD functionality are not included yet.

## Current Features

- Responsive blog page layout
- Clean article/card-style content presentation
- HTML structure for blog content
- CSS styling for visual design
- JavaScript for basic front-end interactivity where needed

## Technologies Used

- HTML
- CSS
- JavaScript

## Planned Improvements

- Add a backend API using Node.js and Express
- Add database storage for posts
- Add create, edit, and delete functionality
- Add comments
- Add user authentication
- Improve accessibility and mobile responsiveness

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/ALEXKk-ai/blog.git
   ```

2. Open the project folder:

   ```bash
   cd blog
   ```

3. Open `index.html` in your browser.

No backend server or database setup is required for the current version.
